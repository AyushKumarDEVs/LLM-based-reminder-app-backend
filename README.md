# LLM-Based Reminder Web App ⏰ (Backend)

## Description

The **LLM-Based Reminder Web App** is an AI-powered tool designed to simplify your daily planning. Instead of manually setting reminders, users can input natural language text, and the app intelligently extracts the **title, description, time, and date** to create an alarm. This eliminates the frustration of manual scheduling, making it easier to stay organized.

## Features ✨

- **AI-Powered Reminder Extraction** 🧠: Uses Gemini LLM to parse user text and extract key details.
- **Seamless Scheduling** 📅: Automatically sets alarms based on detected time and date.
- **Google Calendar Integration** 📆: Sync reminders with Google Calendar for better scheduling.
- **User Authentication** 🔐: Secure login using Google Auth.

## Tech Stack 🛠️

### Frontend

- **React (Vite)** ⚛️ – Fast and lightweight development framework
- **Tailwind CSS** 🎨 – Modern and responsive styling
- **React Router** 🛤️ – Smooth navigation between pages

### Backend

- **Node.js & Express** 🚀 – Scalable server-side architecture
- **MongoDB** 🗄️ – NoSQL database for storing reminders
- **Google Auth** 🔑 – Secure authentication using Google Sign-In
- **Google Calendar API** 📆 – Integration for seamless scheduling
- **Gemini API** 🤖 – LLM-powered text processing for extracting reminder details
